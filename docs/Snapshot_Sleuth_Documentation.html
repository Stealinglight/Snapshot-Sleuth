The Forensic Engine

Snapshot Sleuth leverages Python-based Lambda functions to perform rapid triage analysis. Each scanner runs in isolation, allowing for concurrent processing of evidence.

YARA Scanner

yara_scanner.py

Uses pattern matching to identify malware families, webshells, and suspicious strings within files. Compiles rules dynamically on cold start.

ClamAV

clamav_scanner.py

Standard anti-virus signature scanning. Best for detecting known commodity malware and common attack tools left by adversaries.

Evidence Miner

evidence_miner.py

Heuristic scanner looking for high-value artifacts: SSH keys, AWS credentials, bash history, and sensitive config files.

Extensible Adapters

The packages/adapters module implements a Factory Pattern, allowing you to easily swap or chain notification providers without changing the core logic.

import { AdapterFactory } from './factory'; // Initialize adapters based on config const adapters = AdapterFactory.createAdapters([ 'SLACK', 'GITHUB', 'EMAIL' ]); // Broadcast findings to all channels await Promise.all( adapters.map(adapter => adapter.sendNotification({ title: `Case #${caseId} Analysis Complete`, severity: 'HIGH', findings: scanResults }) ) );
